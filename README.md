# creamyinn

Modern static marketing site for Cream More Ice Cream on `creamyinn.app`.

## What's new

- richer single-page layout with cleaner sections and stronger mobile UX
- data-driven menu cards powered by `assets/menu.json`
- inquiry planner that builds WhatsApp and email-ready order messages
- store-status logic, review slider, FAQ accordion, and navigation polish
- GitHub Pages friendly SEO and PWA support with `robots.txt`, `sitemap.xml`, manifest, favicon, and service worker

## Project structure

- `index.html`: page structure and SEO metadata
- `assets/styles.css`: visual system and responsive layout
- `assets/app.js`: interactivity and client-side logic
- `assets/menu.json`: structured menu content
- `assets/favicon.svg`, `assets/social-card.svg`: branding assets
- `site.webmanifest`, `sw.js`: installability and simple offline caching

## Deployment

This repo is ready for GitHub Pages hosting with the existing `CNAME` file for `creamyinn.app`.
